<div>Controls:</div>
<div>DPAD_UP = Open / Scroll Up</div>
<div>DPAD_DOWN = Scroll Down</div>
<div>DPAD_RIGHT = Move Toggle Right</div>
<div>DPAD_LEFT = Move Toggle Right</div>
<div>A_BUTTON = Select</div>
<div>LB_BUTTON = Back / Exit</div>
